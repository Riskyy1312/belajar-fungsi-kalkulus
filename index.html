Modul Pembelajaran Utama: Fungsi Kuadrat

Materi lengkap dari pengertian dasar, karakteristik grafik, hingga prosedur menggambar parabola dengan contoh pembahasan detail.

1. Pengertian dan Bentuk Umum

Fungsi adalah suatu relasi yang menghubungkan domain (daerah asal) dan kodomain (daerah kawan). Salah satu jenis fungsi yang dipelajari dalam matematika adalah fungsi kuadrat, yaitu fungsi yang mengandung unsur kuadrat atau pangkat dua. Di dalam grafik koordinat Kartesius, fungsi kuadrat selalu membentuk kurva melengkung simetris yang dinamakan parabola.

f(x) = ax² + bx + c   atau   y = ax² + bx + c
⚠️
Syarat Utama
Nilai a, b, c merupakan bilangan real, dan a ≠ 0. Jika a = 0, maka suku kuadrat akan hilang dan fungsi berubah menjadi fungsi linear (garis lurus).

Komponen Pembentuk

a
Koefisien x²
Menentukan arah buka kurva parabola (ke atas atau ke bawah).
b
Koefisien x
Memengaruhi posisi horizontal sumbu simetri grafik.
c
Konstanta
Titik potong grafik terhadap sumbu Y di koordinat (0, c).

2. Karakteristik Grafik Parabola

Karakteristik visual, arah kelengkungan, dan posisi grafik fungsi kuadrat dapat dideteksi melalui kombinasi nilai koefisien a dan nilai Diskriminan (D).

D = b² − 4ac

A. Pengaruh Nilai a (Arah Buka Kurva)

+
a > 0 (Positif)
Parabola terbuka ke atas (senyum 😊). Memiliki titik balik minimum.
−
a < 0 (Negatif)
Parabola terbuka ke bawah (sedih ☹️). Memiliki titik balik maksimum.
💡
Tips Mengingat
Semakin besar |a|, kurva semakin sempit. Semakin kecil |a| mendekati nol, kurva semakin lebar.

B. Pengaruh Nilai b (Letak Sumbu Simetri)

  • a dan b bertanda sama → sumbu simetri di kiri sumbu Y
  • a dan b berbeda tanda → sumbu simetri di kanan sumbu Y
  • b = 0 → sumbu simetri berimpit dengan sumbu Y

C. Pengaruh Diskriminan (D)

D > 0
Memotong sumbu X di 2 titik
a > 0: Puncak di bawah sumbu X
a < 0: Puncak di atas sumbu X
D = 0
Menyinggung sumbu X di 1 titik
Puncak menempel tepat di sumbu X.
a > 0: minimum = 0
a < 0: maksimum = 0
D < 0
Tidak menyentuh sumbu X
Definit Positif (a > 0): melayang di atas
Definit Negatif (a < 0): tenggelam di bawah

3. Sumbu Simetri dan Nilai Optimum

Setiap grafik fungsi kuadrat memiliki titik balik tunggal atau titik puncak ekstrem (xp, yp).

xp
Sumbu Simetri
Garis vertikal yang membagi parabola menjadi dua bagian simetris.
Rumus: xp = −b / (2a)
yp
Nilai Optimum
Nilai puncak tertinggi (maksimum) atau terendah (minimum).
Rumus: yp = −D / (4a)
💡
Tips Praktis
Untuk menghindari kesalahan tanda negatif, hitung xp dulu dengan rumus −b/(2a), lalu substitusikan langsung ke fungsi asalnya: yp = f(xp). Cara ini lebih aman dan cepat!

4. Teknik Menyusun Persamaan

Tergantung informasi titik yang disajikan, ada tiga metode aljabar untuk menyusun persamaan fungsi kuadrat:

1
Dua Titik Potong Sumbu X
Gunakan saat grafik diketahui memotong sumbu X di A(x₁, 0) dan B(x₂, 0) serta melalui satu titik bantu.
y = a(x − x₁)(x − x₂)
2
Titik Puncak (xp, yp)
Gunakan saat diketahui koordinat puncak P(xp, yp) dan satu titik bantu lainnya.
y = a(x − xp)² + yp
3
Tiga Titik Sembarang
Gunakan saat grafik melewati tiga titik acak. Substitusi ketiga titik untuk membentuk SPLTV.
y = ax² + bx + c

5. Prosedur Menggambar Grafik

Ikuti 4 langkah terstruktur berikut untuk menggambar sketsa parabola secara manual dengan akurat:

  1. Titik Potong Sumbu X Substitusikan y = 0, lalu selesaikan persamaan kuadrat untuk mendapatkan akar x₁ dan x₂. Titik potongnya di (x₁, 0) dan (x₂, 0).
  2. Titik Potong Sumbu Y Substitusikan x = 0. Nilai y yang diperoleh sama dengan konstanta c, menghasilkan titik potong (0, c).
  3. Titik Puncak (xp, yp) Hitung sumbu simetri xp = −b / (2a), lalu cari yp dengan rumus diskriminan atau yp = f(xp).
  4. Hubungkan Titik-Titik Plot semua titik yang telah ditemukan ke diagram Kartesius, lalu hubungkan dengan garis melengkung halus hingga membentuk parabola simetris.

6. Aturan Translasi Grafik

Grafik fungsi kuadrat dapat digeser dari grafik dasar f(x) = x² menggunakan rumus kanonik y = a(x − h)² + k:

→
Geser Kanan h satuan
Ganti x dengan f(x − h)
←
Geser Kiri h satuan
Ganti x dengan f(x + h)
↑
Geser Atas k satuan
Tambahkan f(x) + k
↓
Geser Bawah k satuan
Tambahkan f(x) − k

7. Contoh Soal dan Pembahasan

Contoh 1: Menyusun Fungsi dari Dua Titik Potong Sumbu X

Tentukan persamaan fungsi kuadrat yang memotong sumbu X di A(−1, 0) dan B(3, 0) serta melewati titik (0, 3)!

Pembahasan:
Gunakan Metode 1: y = a(x − x₁)(x − x₂)

Substitusi x₁ = −1 dan x₂ = 3:
y = a(x + 1)(x − 3)

Substitusi titik (0, 3) untuk mencari a:
3 = a(0 + 1)(0 − 3) = a(1)(−3) = −3a
a = −1

Substitusi kembali: y = −1(x + 1)(x − 3) = −x² + 2x + 3
Jawaban: f(x) = −x² + 2x + 3
Contoh 2: Menyusun Fungsi dari Titik Puncak

Tentukan rumus fungsi kuadrat yang memiliki titik puncak P(1, −9) dan melewati titik (−2, 0)!

Pembahasan:
Gunakan Metode 2: y = a(x − xp)² + yp

Substitusi xp = 1, yp = −9:
y = a(x − 1)² − 9

Substitusi titik (−2, 0):
0 = a(−2 − 1)² − 9
0 = a(−3)² − 9 = 9a − 9
a = 1

y = (x − 1)² − 9 = x² − 2x − 8
Jawaban: f(x) = x² − 2x − 8
Contoh 3: Menyusun dari Tiga Titik Sembarang

Tentukan fungsi kuadrat yang melewati titik A(2, 7), B(−6, 7), dan C(1, 0)!

Pembahasan:
Gunakan Metode 3: y = ax² + bx + c

Substitusi ketiga titik:
A(2, 7) → 4a + 2b + c = 7 … (1)
B(−6, 7) → 36a − 6b + c = 7 … (2)
C(1, 0) → a + b + c = 0 … (3)

Eliminasi (2) − (1): 32a − 8b = 0 → b = 4a … (4)
Eliminasi (1) − (3): 3a + b = 7 … (5)

Substitusi (4) ke (5): 3a + 4a = 7 → a = 1
b = 4(1) = 4
c = 0 − 1 − 4 = −5

Jawaban: y = x² + 4x − 5
Contoh 4: Kasus Nilai Ekstrem Maksimum

Tentukan parabola yang melalui titik (0, 6) dan (−2, 6) serta memiliki nilai maksimum 8!

Pembahasan:
Dari titik (0, 6): c = 6
Dari titik (−2, 6): 4a − 2b + 6 = 6 → b = 2a … (1)

Nilai maksimum yp = 8:
(b² − 4ac) / (−4a) = 8
(b² − 24a) = −32a → b² + 8a = 0

Substitusi b = 2a: (2a)² + 8a = 0
4a² + 8a = 0 → 4a(a + 2) = 0

a = 0 (tidak memenuhi) atau a = −2
b = 2(−2) = −4

Jawaban: f(x) = −2x² − 4x + 6
Contoh 5: Prosedur Menggambar Sketsa

Uraikan titik-titik koordinat untuk menggambar grafik f(x) = −2x² − 4x + 6!

Pembahasan:

Langkah A: Titik potong sumbu X (y = 0)
−2x² − 4x + 6 = 0 → x² + 2x − 3 = 0
(x + 3)(x − 1) = 0 → (−3, 0) dan (1, 0)

Langkah B: Titik potong sumbu Y (x = 0)
y = 6 → (0, 6)

Langkah C: Titik puncak
xp = −(−4) / (2 × −2) = −1
yp = f(−1) = −2(1) + 4 + 6 = 8 → (−1, 8)

Kesimpulan: Plot titik (−3, 0), (1, 0), (0, 6), dan puncak (−1, 8), lalu hubungkan dengan kurva parabola melengkung ke bawah.
